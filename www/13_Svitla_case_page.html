Business Challenge
The client needed Svitla to create an automated testing tool to efficiently cover all testing scenarios for different types of Ingenico solutions under different stress loads.
Solution:
The Svitla team suggested and Ingenico accepted to use the Cucumber framework. Once Ingenico provided initial training on their solution to Svitla's core team, our specialists used Ruby on Rails to write the necessary code for automated testing.
Value delivered
Reduced time of getting reports from data aggregation site
Well-balanced and simplified workflow for testing new point-of-sale applications
Manual testing process reduced in half by using automation testing
